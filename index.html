လိၵ်ႈတႆး

  • ပိုၼ်ႉထၢၼ်
  • ၽိုၵ်းလူဢၢၼ်ႇ
  • ဢပုမ်ႇ

တူဝ်မႄႈလိၵ်ႈတႆး 19 တူဝ်

ၵ ဝႃႈၵႆႇ ၵွပ်ႈတၢႆႇဝႆႉၼိူဝ်ၵွၼ်း

ၶ ဝႃႈၶႆႇ မႄႈမၼ်းပႆႇမႃးငွမ်း

င ဝႃႈငူး လိူၼ်ႈမႃးၸူးတူဝ်ၶဵတ်ႇ

ၸ ဝႃႈၸင်ႉ ပၢႆဢိူင်ႇဢင်ႉၸွမ်းတၢင်း

သ ဝႃႈသႅင် ငဝ်းသီလႅင်မပ်ႈႁိူဝ်ႈ

ၺ ဝႃႈၺွင်ႇမီးၽၢႆႇႁွင်ႇၵွင်းမူး

တ ဝႃႈတဝ်ႇ တီႈယူႇၵဝ်ႇႁိမ်းၶၢႆး

ထ ဝႃႈထၢႆ တမ်းယႆဝႆႉသၢမ်မၵ်ႈ

ၼ ဝႃႈၼူ မေႃးၸူၸူၸွမ်းၸွၵ်ႉ

ပ ဝႃႈပႃ လႄႇလူၺ်းႁႃၵိၼ်ယိူဝ်ႇ

ၽ ဝႃႈၽိုင်ႈ ၼမ်ႉဝၢၼ်ယိုင်ႈႁူဝ်ႁူင်း

ၾ ဝႃႈၾႃႉ ၼိူဝ်ၾႃႉမီးၵၢင်ႁၢဝ်

မ ဝႃႈမႃႉ တူဝ်ၵျ မီးသဵၼ်ႈလဵဝ်

ယ ဝႃႈယုင်း လႄႇႁႃၸူးၼိူဝ်ႉပိူၼ်ႈ

ရ ဝႃႈရႃႉ မၼ်းၵေႃႈပဵၼ်ရႃႉ

လ ဝႃႈလိင်း မၼ်းၵေႃႈပဵၼ်လိင်း

ဝ ဝႃႈဝႅၼ် ၵေႃႈပဵၼ်ဝႅၼ်ၼၼ်ႉၼႃႇ

ႁ ဝႃႈႁိင်ႇ ပီႈဢဵမ်းႁွႆႈၶေႃးဝူဝ်း

ဢ ဝႃႈဢဝ် ဢဝ်ၽူဝ်ဢဝ်မေး

မႄႈၵပ်းငဝ်ႈ 10 တူဝ်

ဢ လၢႆလၢႆဝႃႈ ဢႃႉ

(ဢ + ဢႃၶျႃး) = ဢႃ

(ဢ + တင်ႇ) = ဢိ

(ဢ + တင်ႇၶတ်ႈၵႅၼ်) = ဢီ

(ဢ + တုတ်ႉၼိုင်ႈ) = ဢု

(ဢ + တုတ်ႉသွင်) = ဢူ

(ဢေသႂ်ႇ + ဢ) = ဢေ

(ဢႄသႂ်ႇ + ဢ) = ဢႄ

(ဢေသႂ်ႇ + ဢ + ဢႃ) = ဢေႃ

(ဢ + တုတ်ႉ-တင်ႇ + ဝ-ၽႅတ်း) = ဢိုဝ်

(ဢ + တုတ်ႉသွင်-တင်ႇ + ဝ-ၽႅတ်း) = ဢိူဝ်

(ဢ + တုတ်ႉသွင် + ဝ-ၽႅတ်း) = ဢူဝ်
